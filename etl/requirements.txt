# 3. Install TA-Lib (REQUIRED for technical analysis):
#    wget http://prdownloads.sourceforge.net/ta-lib/ta-lib-0.4.0-src.tar.gz
#    tar -xzf ta-lib-0.4.0-src.tar.gz
#    cd ta-lib/
#    ./configure --prefix=/usr
#    make
#    sudo make install
#    cd ..
#    rm -rf ta-lib ta-lib-0.4.0-src.tar.gz

# 4. Create virtual environment (recommended):
#    python3.10 -m venv venv
#    source venv/bin/activate
# 
# 5. Install Python packages:
#    pip install --upgrade pip

# ü™ü WINDOWS Installation:
# =======================
# 
# 1. Install Python 3.10+ from https://python.org
#    - Check "Add Python to PATH" during installation
#    - Install pip if not included
# 
# 2. Install Visual Studio Build Tools:
#    - Download from: https://visualstudio.microsoft.com/visual-cpp-build-tools/
#    - Install "C++ build tools" workload
#    - Or install Visual Studio Community with C++ development tools
# 
# 3. Install PostgreSQL:
#    - Download from: https://postgresql.org/download/windows/
#    - Install with default settings
#    - Remember the password for 'postgres' user
# 
# 4. Install Java 11+ (for Spark):
#    - Download from: https://adoptium.net/
#    - Install and set JAVA_HOME environment variable
# 
# 5. Install TA-Lib (REQUIRED for technical analysis):
#    # Method 1: Using pre-compiled wheel (recommended)
#    pip install --find-links https://www.lfd.uci.edu/~gohlke/pythonlibs/ ta-lib
#    
#    # Method 2: Using conda (if you have Anaconda/Miniconda)
#    conda install -c conda-forge ta-lib
#    
#    # Method 3: Build from source (advanced users)
#    # Download TA-Lib source and build with Visual Studio
# 
# 6. Open Command Prompt or PowerShell as Administrator:
#    # Create virtual environment (recommended)
#    python -m venv venv
#    venv\Scripts\activate
#    
#    # Upgrade pip
#    python -m pip install --upgrade pip
#    
#    # Install packages
#    pip install -r requirements.txt
# 
# 7. Verify installation:
#    python -c "import streamlit, pandas, psycopg2, vnstock, ta; print('‚úÖ All packages installed successfully')"
# 
#

# üîß TROUBLESHOOTING:
# ===================
# 
# Common Issues and Solutions:
# 
# 1. TA-Lib Installation Error:
#    - Linux: Install build-essential first, then build TA-Lib from source
#    - Windows: Use pre-compiled wheel from lfd.uci.edu
#    - macOS: Use Homebrew to install ta-lib
# 
# 2. PostgreSQL Connection Error:
#    - Ensure PostgreSQL is running
#    - Check connection parameters in .env file
#    - Verify firewall settings
# 
# 3. VNStock API Error:
#    - Update vnstock: pip install --upgrade vnstock
#    - Check internet connection
#    - Verify API endpoints
# 
# 4. Memory Issues:
#    - Increase virtual memory (Windows)
#    - Use swap file (Linux)
#    - Close unnecessary applications
# 
# 5. Permission Errors (Linux/macOS):
#    - Use virtual environment
#    - Avoid using sudo with pip
#    - Check file permissions
#
