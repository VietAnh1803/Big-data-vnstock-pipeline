# Images Directory

Thư mục này chứa các hình ảnh cho documentation.

## Files

- `system-architecture.png` - Sơ đồ kiến trúc hệ thống (Real-time Stock Analysis System Architecture)

## Cách thêm ảnh

1. Đặt file ảnh vào thư mục này
2. Trong README.md, sử dụng cú pháp:
   ```markdown
   ![Alt text](images/filename.png)
   ```

## Lưu ý

- Sử dụng định dạng PNG hoặc SVG cho diagrams
- Tối ưu kích thước file để tránh làm chậm khi load README trên GitHub
- Tên file nên rõ ràng và mô tả nội dung
